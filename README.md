# POST IA Landing

Landing pública de POST IA preparada para GitHub Pages.

## Publicar

1. Crea un repositorio público llamado `post-ia-landing`.
2. Sube todos los archivos de esta carpeta conservando la carpeta `assets`.
3. Abre `Settings > Pages`.
4. En `Source`, selecciona `Deploy from a branch`.
5. Selecciona la rama `main` y la carpeta `/(root)`.
6. Pulsa `Save`.

La dirección será:

`https://TU-USUARIO.github.io/post-ia-landing/`

La publicación puede tardar algunos minutos.

## Archivos públicos

- `index.html`
- `styles.css`
- `script.js`
- `.nojekyll`
- `assets/post-ia-agent-hero.png`
- `assets/mode21-barberia.png`

El formulario está conectado con la aplicación web de Google Apps Script configurada en `script.js`.
